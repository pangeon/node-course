# node-course
Basics of nodejs. Small and simple applications.

# deployment guide
1) Instal nodejs from [Node JS Homepage](https://nodejs.org/en/download/)
2) Most application haven't GUI. Run them in console.
3) Download dependecies via npm:

```
npm install

```
